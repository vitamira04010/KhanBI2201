## О репозитории
Данный репозиторий создан в учебных целях для изучения Git и GitHub.
В частности, он демонстрирует:
*   Создание и настройку репозитория.
*   Работу с ветками (`feature-readme`).
*   Базовые команды Git (`add`, `commit`, `push`).
*   Оформление документации с помощью **Markdown**.
